Vite es una herramienta de construcción y servidor de desarrollo muy rápida, pensada para proyectos modernos con ES modules. En este curso se recomienda como opción principal para crear proyectos React con TypeScript.

## Crear proyecto con TypeScript

```bash
npm create vite@latest mi-app -- --template react-ts
cd mi-app
npm install
npm run dev
```

El template **react-ts** incluye TypeScript, `tsconfig.json`, `vite.config.ts` y archivos de entrada `.tsx` por defecto.

## Estructura típica

- index.html en la raíz (punto de entrada HTML).
- src/main.tsx: entrada de la aplicación (createRoot y render).
- src/App.tsx: componente raíz.
- tsconfig.json, vite.config.ts: configuración de TypeScript y Vite.

## Scripts

- `npm run dev`: servidor de desarrollo.
- `npm run build`: build de producción.
- `npm run preview`: previsualizar el build.

Los componentes usan extensión `.tsx` cuando contienen TSX. Otras opciones con TypeScript en este material: Create React App (`--template typescript`, 7.1) o proyecto desde 0 con Webpack+Babel+TS (8.1).
